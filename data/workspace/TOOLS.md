# TOOLS.md - 本機筆記本

Skills 定義工具的「運作方式」。此檔案用於記錄「您的特定資訊」—— 專屬於您設定的內容。

## 此處記錄什麼

例如：

- 攝影機名稱和位置
- SSH 主機和別名
- TTS 偏好語音
- 喇叭/房間名稱
- 裝置暱稱
- 任何環境特定內容

## 範例

```markdown
### 攝影機

- living-room → 主要區域，180° 廣角
- front-door → 入口，動態觸發

### SSH

- home-server → 192.168.1.100，使用者：admin

### TTS

- 偏好語音：「Nova」（溫暖，略帶英式口音）
- 預設喇叭：廚房 HomePod
```

## 為什麼要分開？

Skills 是共享的。您的設定是您自己的。將它們分開意味著您可以更新 skills 而不會失去筆記本，並且可以分享 skills 而不會洩露您的基礎設施。

新增任何有助於您工作的內容。這是您的速查表。
